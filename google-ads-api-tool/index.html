Google Ads Campaign Workflow Tool

An internal workflow, validation and reporting application for authorized Google Ads accounts. Developed by independent developer Lian Shuai.

About The Tool

This project supports internal Google Ads campaign preparation, validation, deployment and reporting workflows. It does not replace Google Ads or Google Ads Editor.

The tool is designed to connect structured campaign workflow steps for authorized Google Ads accounts only.

It is not offered as a public self-service advertising platform, automated advertising robot, or Google product.

Supported Capabilities

  • Campaign Creation
  • Campaign Management
  • Reporting

Supported Campaign Type

Search Campaigns

Workflow

A controlled process for preparing, validating and reporting on authorized campaigns

  1. Campaign Preparation
  2. Quality Validation
  3. Authorized Review
  4. Google Ads API
  5. Campaign Deployment
  6. Performance Reporting

Security

  • OAuth 2.0 authorization is used for account access.
  • Google passwords are not stored by this website or tool.
  • Only authorized Google Ads accounts are supported.
  • Access is limited to the minimum Google Ads API permissions needed for campaign workflow and reporting.
  • The tool does not use scraping.
  • The tool does not support unauthorized access.

Data Use

Google Ads API data is used to prepare Search Campaign structures, validate campaign settings, submit authorized campaign changes and generate internal performance reports.

Google Ads data is not sold and is not used for unrelated advertising or profiling outside the stated workflow.
