Developer Focus

Internal Advertising Workflow Tools

The developer project area on MenswearStyle.store documents an internal Google Ads campaign workflow tool. The tool is focused on practical campaign operations: preparing Search Campaign structures, validating required campaign fields, reviewing authorized changes and generating reporting for approved Google Ads accounts.

Project Boundary

Google Ads API Usage

The project supports campaign creation, campaign management and reporting for Search Campaign workflows. OAuth 2.0 is used for authorization, and account access is limited to Google Ads accounts explicitly authorized by permitted users.

The Google Ads API developer token application should identify this project as a Google Ads Independent Developer application, with google-ads-api@menswearstyle.store as the domain-aligned API contact email.

Editorial Website

Menswear Content

MenswearStyle.store remains a menswear review and style guide website. The developer project area is added as a clearly separated section so Google Ads API reviewers and users can understand the tool purpose, privacy practices and contact channel.

Application Alignment

The application should use Lian Shuai as the developer identity, Google Ads Independent Developer as the company type, and https://menswearstyle.store/google-ads-api-tool/ as the application homepage.

Access Boundary

No Public Account Access

The tool is not offered as a public SaaS, agency platform, or account resale service. It is limited to accounts owned by the developer or accounts where valid authorization has been granted.
